# Sample Problem

X 40인자(binary 32 / ordinal 4 / categorical 4)와 반응 Y=[y11,y12,y21,y22]에서 **ΣY를 최대화하는 X**를 찾는 이산·혼합 최적화 문제.
목적함수는 데이터셋이 아니라 **ground truth 비밀식 자체**(노이즈 없는 결정적 함수)를 직접 평가한다(A 패러다임 = 새 조건 실험이 가능한 상황).

## Dataset / Case 설명

동일한 X 구조에 난이도를 단계적으로 키운 **Case1~6** (Case1 최저 → Case6 최고).
복잡도 축: main effect 수↑, 고차 결합↑(2차→quadratic→3차→**4차, Case6**), 노이즈/outlier↑ (단, 노이즈는 CSV 생성용일 뿐 A 최적화엔 무관).

## 복잡도 계산방식 설명

6개 지표(총 항수, 최대 교호차수, 고차(≥3) 개수, 선형근사 gap%, 국소최적 개수, trap 비율)를 0~100으로 정규화·평균한 **종합 복잡도 지수**.
핵심: 항 수보다 **다봉성(국소최적·trap)**이 난이도를 좌우 — 항이 적어도 다봉인 Case3가 가장 어렵다.

## 결과

**EVAL(평가) 예산**에선 BO가 최강(500평가로 ~0% gap), GA·SA도 거의 전역최적; vanilla TPE는 변수독립 가정 탓에 부진.
**TIME(시간) 예산**에선 순위 역전 — 같은 시간에 수만 번 평가하는 **GA·PSO가 승리**, GP/RF 적합에 시간을 쓰는 BO·SMAC은 급락("샘플효율 ≠ 시간효율").
